How modders will be able to create gamemodes in Physics using C#

Feburary 17, 2021 by Cooper J.

My Goals

I don't want making addons and gamemodes to be a pain. I want you to be able to make a simple C# script, load it in, and play it.

Here are my goals

  • Load new code at runtime, loads in quickly
  • Hotloads code while playing
  • Easy to learn and start scripting
  • You don't have to install anything other than the base game (no SDks)

Console

In the last blog post I talked a bit about how I wanted a console in-game. I started working on it and I'm about 50% done.

This page gives you info for the gamemode. On the left is the gamemode json file, on the right is the stuff in the gamemode (or just currently open ingame). You can refresh to see any changes you've made.

Here's the main console window. You can enter commands, and eventually see logs. The only commands so far are input, which displays the keybind info, and map [mapname], which loads a map. I don't want custom commands, as I fear that might mess up the game for some players. But it might happen someday.
