Devblog 1 - C#

How modders will be able to create addons with C#

My Goals

I don't want making addons and gamemodes to be a pain. I want you to be able to make a simple C# script, load it in, and play it. Here are my goals:

  • Load new code at runtime, loads in quickly
  • Hotloads code while in-game
  • Easy to learn and start scripting
  • You don't have to install anything other than the base game (no SDks)

Console

In the last blog post I talked a bit about how I wanted a console in-game. I started working on it and I'm about 50% done. Here's the main console window. You can enter commands, and eventually see logs. The only commands so far are input, which displays the keybind info, and map [mapname], which loads a map. I don't want custom commands, as I fear that might mess up the game for some players. But it might happen someday.
