July Update

Release recap and general updates

Recap

The first main update was a couple of days after the Early Access release, where I added the BlockBuilder gamemode. This gamemode was broken in release, so I completely revamped it. It's not as good as I want it to be but I still like it.

I also noticed that most players were having a problem where they didn't know the gamemode-specific keybinds. I've been working on adding a settings menu where people can see all this stuff.

Some achievements have been fixed like the time waster and builder achievement.

I've been working on revamping the UI, starting with the menu. It was pretty bare before. Now gamemodes have icons and there's more options.

Public Dev Branch

I pushed a branch on steam that's pretty much a beta build. You can see most of the new features on there as soon as they're added. It includes the updated UI and new addons (BlockBuilder and Pool). Keep in mind it updates more than the main branch and can be unstable.

Pool

Pool is a simple gamemode I added with the dev branch. I wanted to make a quick gamemode using a different player controller. It's not exactly pool but it's still fun.
