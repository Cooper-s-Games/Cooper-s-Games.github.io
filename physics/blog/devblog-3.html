Devblog 3 - UI

How I'm going to use HTML to create easy-to-modify UI

Why HTML?

I know HTML UI can sound hard to mess with at first, but it's actually pretty simple, and much easier to edit than UE's widgets. Putting all this work into UI allows it to be easier to the future, and what I want with my game is for modding to be easy and fun to mod.

Components

I want to create a interface similar to s&box, so I'm gonna use vue components to display stuff. I made a health/ammo bar.
