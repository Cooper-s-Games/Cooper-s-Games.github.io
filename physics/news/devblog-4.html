Some stuff that I've been working on

18 April 2021

Developer access

I've given developer access out to a few people. It's been really helpful to see a bunch of bugs to fix and features to add. I was previously working on a bunch of features that the game really didn't need, so it's been good to get back on track and work on stuff for both the future players and developers.

Suicide Barrels

I've been working on a Suicide Barrels gamemode to test AI and ragdolls.

I'm using most code from my deathmatch base, so it's been pretty simple to code.

I'll probably release the code later on, so people can get some more examples of how gamemodes will be made. While I was testing I made some bots that just run around and shoot/explode randomly

Deathmatch

Deathmatch is still pretty raw. I've been using it as the base shooter gamemode, but I know there's still a bit of work I need to do on it to make it more deathmatch-y and less devtesting-y.

Custom Imports

Models from Source, are of course, different sizes and have different bones. For example, mossman is a giant and I had to scale her down to 0.025 to make it look normal. So I've been expirementing with custom asset import sizes.

You'll be able to have a bunch of options to mess around with. You can even leave it as the default import if you want.
