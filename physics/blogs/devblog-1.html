Dev Blog 1 - C#

My Goals
I don't want making addons and gamemodes to be a pain. I want you to be able to make a simple C# script, load it in, and play it. Here are my goals:
  • You can actually use c#
  • Load new code at runtime, loads in quickly
  • Hotloads code while in-game
  • Easy to learn and start scripting
  • You don't have to install anything other than the base game (no SDks)
