Importers
You (most likely) will be able to write your own importers. I want to keep the base importers open source, so people can check them out and make their own. I don't want to create a custom file format, since that would be tougher to import files with. You will hopefully be able import the usual file types: bsp, fbx, obj, and mdl. The Goal
My goal, which is a stupid one, is to have a good base importer, and allow people to make their own importers and share them, and have a bunch of different importers so people who don't know how to program can just load a script somebody made, import stuff onto their game, then play it with their friends.
